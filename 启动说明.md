# 启动说明

## 快速开始

### 1. 安装Node.js依赖
```bash
npm install
```

### 2. 安装Python工具（用于YouTube视频转换）

**macOS:**
```bash
# 安装yt-dlp（推荐）
pip3 install yt-dlp

# 安装ffmpeg（用于音频转换）
brew install ffmpeg
```

**Linux:**
```bash
# 安装yt-dlp
pip3 install yt-dlp

# 安装ffmpeg
sudo apt-get update
sudo apt-get install ffmpeg
```

**Windows:**
```bash
# 安装yt-dlp
pip install yt-dlp

# 下载ffmpeg并添加到PATH
# https://ffmpeg.org/download.html
```

### 3. 启动服务器
```bash
npm start
```

### 4. 打开浏览器
服务器启动后会自动提示地址，通常是：
```
http://localhost:3000/index.html
```

## 功能说明

- ✅ 完整的YouTube视频转音频功能
- ✅ WebGL音频可视化
- ✅ 音频播放和下载
- ✅ 本地存储管理

## 故障排除

### 如果提示"未找到yt-dlp"
```bash
pip3 install yt-dlp
# 或
pip install yt-dlp
```

### 如果提示"未找到ffmpeg"
**macOS:**
```bash
brew install ffmpeg
```

**Linux:**
```bash
sudo apt-get install ffmpeg
```

### 如果端口被占用
修改 `server.js` 中的 `PORT` 变量（默认3000）
